# Password Generator

A simple password generator tool in Python that creates secure passwords using a mix of uppercase letters, lowercase letters, digits, and special characters.

## Overview

This Python script generates random passwords based on user-defined length. It creates strong passwords by combining characters from various sets including uppercase letters, lowercase letters, digits, and punctuation. Users can choose the length of the password they want to generate and also have the option to record the password in a text file.

## Features

- Generates random passwords of the specified length.
- Allows users to choose the length of the password.
- Offers an option to record the generated password in a text file along with the corresponding social media platform (e.g., Google, Instagram, Twitter, etc.).
- Provides a secure mix of uppercase letters, lowercase letters, digits, and punctuation characters.

## Getting Started

### Prerequisites

- Python 3.XX

### Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/your_username/PasswordGenerator.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Simple-Password-Generator
   ```

3. Run the Python script:

   ```bash
   python3 password_generator.py
   ```

4. Enter the desired length of the password when prompted.
5. Choose whether to record the password in a text file by responding with 'y' or 'n'.
6. If you opt to record the password, input the social media platform for which the password is intended.
7. The generated password will be displayed in the terminal. If recorded, it will also be stored in a `password.txt` file in the project directory.
